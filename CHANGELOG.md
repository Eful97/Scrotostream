# Changelog

All notable changes to ScrotoStream will be documented in this file.

## [1.1.58] - 2026-04-18

### Added
- Formatter unit tests (`test/formatter.test.js`)
- GitHub Actions CI workflow (`.github/workflows/ci.yml`)

### Changed
- **Renamed** from easystreams to **ScrotoStream**
- Package name: `easystreams` → `scrotostream`
- Manifest name: `EasyStreams` → `ScrotoStream`
- Scraper IDs: `easystreams-*` → `scrotostream-*`
- Stremio addon ID: `org.bestia.easystreams` → `org.scrotostream`
- Stremio addon name: `Easy Streams` → `ScrotoStream`

### Fixed
- Updated formatter emojis:
  - 1080p: 🚀 FHD → 👌 FHD
  - 720p: 💿 HD → ⚡ HD
  - Provider icon: 📡 → ⚙️

### Removed
- Reference to old easystreams domain in README
